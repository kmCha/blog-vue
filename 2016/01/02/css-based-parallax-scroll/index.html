纯css实现面向未来的视差滚动

发表于 2016-01-02   |   分类于 css   |  

视差滚动

所谓视差滚动，顾名思义，就是滚动的时候会带有视差效果。

那什么叫有视差效果呢，就是你在滚动的时候，页面的不同部分的滚动速度不一致，导致的有趣的效果。视差滚动这个概念也不是最近才流行起来的，实际上现在很多的网站都使用了这样的一个技术。然而现目前时差滚动基本上都是由javascript来驱动的，为什么他们不用纯css来做呢，原因很简单，要用纯css来做的话，用到的css规则还不在标准之中或者还在标准的草案阶段，浏览器支持不全面，不能照顾老版本的IE浏览器等等。所以，想要在正规一点的项目中支持老版浏览器同时又要实现视差滚动，纯css方法不是现目前的最优选择，因此我说这个方法是“面向未来”的。另外值得一提的是，通过本方法实现的视差滚动相当平滑，并不会阻塞页面正常渲染流水线，因为用了3dtransform的元素会开启GUP渲染，在单独的渲染层渲染，不会触发页面的reflow和repaint。

基本原理

要实现上述效果，我们需要将我们的页面拆成一个3d的空间，而不是我们平时习惯的2d平面。试想一个元素贴在你电脑的屏幕上，另一个元素贴在比你屏幕离你眼睛更近的一个平行的屏幕上，两个平行的屏幕都会随页面的滚动而同步滚动，那么这时候离你更近的那个元素看上去就会比远的那个滚得要快一点，这个现象可以拿近大远小来解释。对，近大远小四个字就可以总结我们纯css实现视差滚动的原理。原理知道了，那怎么用css来实现呢？在详细说明之前先来看看最基本的骨架：

<div class="parallax">
    <div class="parallax-layer-back">
        <center>我是在屏幕后面的元素</center>
    </div>
    <div class="parallax-layer-base">
        <center>我是在屏幕上的元素</center>
    </div>
</div>

下面是css文件：

.parallax {
    perspective: 1px;
    height: 50vh;
    overflow-x: hidden;
    overflow-y: auto;
}
.parallax-layer-base {
    padding: 50vh 0;
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    transform: translateZ(0);
}
.parallax-layer-back {
    padding: 50vh 0;
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    transform: translateZ(-1px);
}

在html文件中，我们用.parallax元素包围了一个.parallax-layer-base和一个.parallax-layer-back元素。接着在最关键的css文件中，给.parallax元素设置了一个perspective: 1px;规则，这个css规则就是实现视差滚动的核心。perspective属性可以在相应的元素上建立起一个3d视口，而他的值就是模拟你的眼睛与他之间的距离，值越大，离得越远，值越小，离得越近。下面来看一张很形象的图来帮助理解，图片来自这里：

这里我们可以看到，浏览器在.parallax元素元素上建立起了一个3d视口，对应上图中的蓝色部分，他是在我们电脑的屏幕上的。而指定了这个属性的元素的后代元素都会以3d的形态投影在这个元素的平面上。上图中黄色的元素就相当于我们的.parallax-layer-back元素，沿着z轴向后移动了1px，也就是沿着z轴平移了－1px，对应图中的z线段。而perspective属性的值是1px，也就是模拟我们眼睛距离视口的长度是1px，对应图中的p线段。既然p＝z，我们可以很容易的得到黄色元素在蓝色元素上的投影会缩小成二分之一大小。看完了原理，我们来看看实际是什么样子的：

我是在屏幕后面的元素
我是在屏幕上的元素

可以看到，.parallax-layer-base元素跟我们的.parallax在一个平面上，而.parallax-layer-back元素在.parallax元素后面1px的位置。因为我们的眼睛距离.parallax元素1px，因此.parallax-layer-back元素投影在.parallax元素上就只有一半那么大，而.parallax-layer-base元素是.parallax-layer-back元素的两倍大。那么我们要想将.parallax-layer-back元素变成看起来跟.parallax-layer-base元素一样大的话，只要将它用scale来放大两倍就行了。

.parallax-layer-back {
    transform: translateZ(-1px) scale(2);
}

通过上面的斜视图可以很清楚的看到，虽然.parallax-layer-back元素在.parallax元素上的投影跟.parallax-layer-base一样大，但是实际上他被放大成了两倍，因此要滚动完他就需要比.parallax-layer-base元素更多的滚动路程，因此看起来滚动得更慢。
在实际的环境中，我们不只会使用仅仅一个背景和base，因此分组很重要，我们可以将同一个中心位置的背景、base和前景（后面介绍）分为一组，然后同时加入多组到.parallax元素中。

html：

<div class="parallax">
    <div class="parallax-group group1">
        <div class="parallax-layer-base base1">
            <center>我是在屏幕上的元素1</center>
        </div>
    </div>
    <div class="parallax-group group2">
        <div class="parallax-layer-back back1">
            <center>我是在屏幕后面的元素1</center>
        </div>
        <div class="parallax-layer-base base2">
            <center>我是在屏幕上的元素2</center>
        </div>
    </div>
    <div class="parallax-group group3">
        <div class="parallax-layer-base base3">
            <center>我是在屏幕上的元素3</center>
        </div>
        <div class="parallax-layer-fore fore1">
            <center>我是在屏幕前面的元素1</center>
        </div>
    </div>
    <div class="parallax-group group4">
        <div class="parallax-layer-back back2">
            <center>我是在屏幕后面的元素2</center>
        </div>
        <div class="parallax-layer-base base4">
            <center>我是在屏幕上的元素4</center>
        </div>
    </div>
    <div class="parallax-group group5">
        <div class="parallax-layer-base base5">
            <center>我是在屏幕上的元素5</center>
        </div>
    </div>
</div>

css：

.parallax {
    perspective: 300px;
    height: 50vh;
    overflow-x: hidden;
    overflow-y: auto;
}
.parallax-group {
    position: relative;
    height: 50vh;
    transform-style: preserve-3d;
}
.parallax-layer-base {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    transform: translateZ(0);
}
.parallax-layer-back {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    transform: translateZ(-300px) scale(2);
}
.parallax-layer-fore {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    transform: translateZ(120px) scale(.6);
}
.group1 {
    z-index: 5;
}
.group2 {
    z-index: 3;
}
.group3 {
    z-index: 4;
}
.group4 {
    z-index: 2;
}
.group5 {
    z-index: 3;
}

在html文件中，我们定义了parallax-group元素来包围每一组元素，在css文件中用相对定位来让其子元素等于他的尺寸。而在子元素中，我们又增加了一个.parallax-layer-fore元素，他与.parallax-layer-back元素相反，在屏幕的上方，因此滚动得更快。至于transform: translateZ(120px) scale(.6);中的放大系数0.6，我们可以用公式scale factor = 1 + (translateZ * -1) / perspective(来自这里)得到。另外，由于.parallax-layer-back元素的实际尺寸是我们.parallax-layer-base元素的两倍，因此为了达到最好的效果最好像上面一样隔一个.parallax-layer-base元素设置一个.parallax-layer-back元素。同时，需要设置好z-index的值确保每一层都能正确的显示。

通过将页面分为多层（本例为3层，实际上可以根据自己喜好或者项目需求任意改写transform: translateZ(xxxpx) scale(xxx);属性来增加任意多的层），并且合理设定z-index值就可以实现视差滚动的效果。
