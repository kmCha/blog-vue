纯CSS实现简洁自定义开关样式

发表于 2015-12-12   |   分类于 css   |  

在做网页的时候，项目里的各个组件外观通常要保证于其他组件的统一，不然看起来会很突兀。在项目中除了提交表单外，input[type="checkbox"]组件还有一个作用就是可以用CSS轻易的”破坏”他的外观然后来定义你想让他成为的样子来适配你项目的主题样式。下面来分享一下实现自定样式的基本原理。

checkbox元素

一个checkbox元素，多数时候被用在表单的提交中用于表示一个键的true或false。他有两个值，选中表示true，没选中表示false。既然知道他能表示true和false，那么用的地方就能够无限的扩展了，比如典型的，一个开关就是两个值。所以表示开关一般也用checkbox，那么他在一个页面中是这样的：
他的外观随不同操作系统而存在一点差异，不过不管什么操作系统，他都是一个框框加上里面一个钩钩而已。在提交表单也就是你要用到他的值的时候，这当然不会存在什么问题。但是你要想用它来提升用户体验的话，最好还是换一下它的样式。什么意思？举个例子，下面是我做的一个项目里的一个组件，我在一个框中的四个边角上放上了四个操作组件，其中左下角是要一个布尔值的组件来表示开关，如果我用系统自带的checkbox的话：

各位感受一下，这个checkbox看起来怎么样，请说实话。

纯CSS实现上述开关样式

上面动图中的开关就是将系统自带checkbox改掉之后的效果，在说代码之前先说一下基本原理。实现这个效果的关键是label元素，每个label元素能够通过id绑定一个checkbox元素，而绑定之后，点击label和点击checkbox都能够触发状态的改变。因此，不能改变自带checkbox的外观的话，那就简单粗暴点直接把它隐藏了，通过css来自定义label的外观就行了。为了实现上面展现的效果，外面必须用个div把checkbox和label包起来，里面的滚动效果用label来实现，外面的包围部分用div的样式来实现。
下面给出包括上述开关在内的几种开关样式的代码，并且在最后附上了组件的实物，可以点击对应的开关查看效果。

HTML代码

<div class="km-switch switchWrap-x">            //x替换为相应的数字
    <input id="switch-x" type="checkbox">
    <label for="switch-x"></label>
</div>

CSS代码

//km-switch.css
.km-switch input[type=checkbox] {
    visibility: hidden;
}

/*switch-1*/
.switchWrap-1 {                                                /*设置包围div的样式*/
    position: relative;
    height: 44px;
    width: 44px;
    border-radius: 6px;
    background-color: #46d4f3;
}
.switchWrap-1:before {                                /*用before伪类添加开关内部表示OFF的"-"*/
    content: "";
    position: absolute;
    top: 10px;
    left: 17px;
    height: 2px;
    width: 10px;
    background-color: #fff;
}
.switchWrap-1:after {                                    /*用after伪类添加开关内部表示ON的点*/
    content: "";
    position: absolute;
    top: 28px;
    left: 18px;
    height: 8px;
    width: 8px;
    border-radius: 100%;
    background-color: #fff;
}
.switchWrap-1 label {                                    /*设置内部label的样式（这里是滑块）*/
    position: absolute;
    left: 0px;
    top: 0px;
    height: 18px;
    width: 40px;
    background-color: #fff;
    border: 2px solid #46d4f3;  /*蓝色*/
    transition: all 0.5s ease;
    border-radius: 6px;
    cursor: pointer;
    z-index: 2;
}
#switch-1:checked + label {                        /*设置选中之后滑块的位置*/
    top: 22px;
}

/*switch-2*/
.switchWrap-2 {                                                /*设置包围div的样式*/
    position: relative;
    height: 44px;
    width: 44px;
    border-radius: 50%;
    background-color: #fff;
    box-shadow: 0 0 5px 1px rgba(0,0,0, 0.4);
}
.switchWrap-2 label {
    position: absolute;
    left: 6px;
    top: 6px;
    height: 32px;
    width: 32px;
    background-color: #ccc;
    transition: all 0.5s ease;
    border-radius: 50%;
    cursor: pointer;
    box-shadow: inset 0 0 1px 1px rgba(0,0,0, 0.2);
}
#switch-2:checked + label {
    background-color: #46d4f3;
}

/*switch-3*/
.switchWrap-3 {                                                /*设置包围div的样式*/
    position: relative;
    height: 22px;
    width: 44px;
    border-radius: 5px;
    background-color: #7f7f7f;
}
.switchWrap-3:before {
    content: "";
    position: absolute;
    top: 9px;
    left: 5px;
    height: 4px;
    width: 34px;
    border-radius: 2px;
    background-color: #444;
}
.switchWrap-3 label {
    position: absolute;
    left: 3px;
    top: 3px;
    height: 16px;
    width: 16px;
    background-color: #fe67c3;  /*粉色*/
    transition: all 0.5s ease;
    border-radius: 50%;
    cursor: pointer;
}
#switch-3:checked + label {
    left: 25px;
    background-color: #46d4f3;
}

/*switch-4*/
.switchWrap-4 {                                                /*设置包围div的样式*/
    position: relative;
    height: 22px;
    width: 44px;
    border-radius: 5px;
    background-color: #7f7f7f;
    overflow: hidden;
}
.switchWrap-4 label {
    position: absolute;
    left: 0px;
    top: 0px;
    height: 22px;
    width: 22px;
    background-color: #fe67c3;
    transition: all 0.5s ease;
    border-bottom-left-radius: 5px;
    border-top-left-radius: 5px;
    border-bottom-right-radius: 0px;
    border-top-right-radius: 0px;
    cursor: pointer;
    z-index: 2;
}
.switchWrap-4 > label:before {
    content: "";
    position: absolute;
    top: 8px;
    left: -14px;
    height: 6px;
    width: 6px;
    background-color: #444;   /*蓝色*/
    border-radius: 50%;
}
.switchWrap-4 > label:after {
    content: "";
    position: absolute;
    top: 10px;
    left: 30px;
    height: 2px;
    width: 6px;
    background-color: #444;  /*粉色*/
    border-radius: 1px;
}
#switch-4:checked + label {
    left: 22px;
    border-bottom-right-radius: 5px;
    border-top-right-radius: 5px;
    border-bottom-left-radius: 0px;
    border-top-left-radius: 0px;
    background-color: #46d4f3;
}

具体解释在代码注释中体现出来了，就不仔细讲了，因为这个概念只要知道实现原理了以后，每个人心里的开关样式都不一样。因此可以用这个原理来做出自己喜欢的组件，不必照搬别人的设计。
